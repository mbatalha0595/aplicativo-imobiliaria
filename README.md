# Imobiliária

## Objetivo
Este aplicativo tem como objetivo permitir a gestão de imóveis de uma imobiliária.

## Telas

### Imóveis
Esta tela exibe imóveis que estão disponíveis, incluindo alguns filtros para simplificar a busca por determinado imóvel conforme a necessidade do cliente.

<img src="https://raw.githubusercontent.com/mbatalha0595/aplicativo-imobiliaria/main/images/Imoveis.jpg" width="750">

### Detalhes do Imóvel
Esta tela exibe detalhes sobre o imóvel selecionado.

<img src="https://raw.githubusercontent.com/mbatalha0595/aplicativo-imobiliaria/main/images/DetalhesImovel.jpg" width="750">

#### Expandir a Imagem da Galeria

Para expandir a imagens, basta clicar na imagem destacada na galeria.

<img src="https://raw.githubusercontent.com/mbatalha0595/aplicativo-imobiliaria/main/images/DetalhesImovelGaleria.jpg" width="750">

#### Imprimir Informações do Imóvel

Se necessário, o usuário pode imprimir ou salvar informações do imóvel em PDF.

<img src="https://raw.githubusercontent.com/mbatalha0595/aplicativo-imobiliaria/main/images/DetalhesImovelImpressao.jpg" width="750">

<img src="https://raw.githubusercontent.com/mbatalha0595/aplicativo-imobiliaria/main/images/DetalhesImovelImpressaoResultado.jpg" width="750">

### Suporte
Esta tela permite que o usuário abra uma chamado de suporte (e-mail), além de permitir anexar um arquivo.

<img src="https://raw.githubusercontent.com/mbatalha0595/aplicativo-imobiliaria/main/images/Suporte.jpg" width="750">
